# Power-Point
This project was developed for practice while learning PowerPoint through an online e-learning platform. It demonstrates how animation can be used to create interactive and visually appealing presentations.







🎯 Why I Created This Project

I created this mini project to practice my PowerPoint skills while learning from an online course. Instead of only watching lessons, I applied what I learned by creating a ship route animation presentation. This project helped me improve my knowledge of animations, transitions, slide design, and visual storytelling. It also gave me hands-on experience in creating interactive and professional presentations.
